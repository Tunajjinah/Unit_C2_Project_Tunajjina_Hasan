Intro to HTML

Paragraph and headings

Paragraphs

Most webpages use <p> to format text. Making it known as the "workhorse" tag of every web page. This tag is a block level element since it creats a space above and below on the page.

However if you want to create a line break you can use the <br> tag.

Lorem Ipsum Text

Lorem Ipsum was invented well in advance of computers in order for book designers to layout their projects prior to receiving the actual text from writers. Web designers currently utilize it for creating page mockups.

Horizontal Rule and Preformatted text

If you want to create a line between blocks of text, use the <hr> element. hr is an empty tag. If you want to preserve multiple spaces or line breaks for text like in poems, you can use the preformatted text <pre> tag. pre is a block-level element.

Headings

Headings are used to create a outline structure on a web page. It not only changes the size of the texts and bolds them but it also allow assistive devices like screen readers. Headings are block level elements.
