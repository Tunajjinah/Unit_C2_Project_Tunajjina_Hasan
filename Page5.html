Images and comments

Comments

HTML comments are typically hidden in the document. n order to create an HTML comment, you must use the characters "<!--" before the comment text and "-->" after it.

What is the purpose of html comments

Comments in html allows developers to leave notes for themselves or for another developer. This can also be used to temporarliy deactivate certain sections for debugging.

Attributes

Attributes are codes that appear within a tag that modify it in some way. The general syntax for an attribute is: <tagName attributeName="value">content</tagName>

Images

Images help improve the appearence of a webpage.

Images are embedded on a web page using the <img> element. It is an empty element (no trailing tag) and works inline. To locate the image, you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.

  1. Local image= <img src="happyface.jpg" alt="Smiley Face">
  2. Remote images = <img src="https://ex.com/hf.jpg" alt="Smiley Face">

Sizing images

In order to size images you can use the width and height attribute.

<img src="file" width="200">

Example of images in html:

  • The scr attribute specifies the path (URL) to the image.
  • alt - Specifies an alternate text for the image
