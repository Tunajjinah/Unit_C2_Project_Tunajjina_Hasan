Links

To add links in html you have to use the anchor / <a> to make a hyperlink to a different page or some othee location on the same website.

In order to locate the link you need to use the href attribute.

  • ethernal links : <a href="https://google.com"> Google</a>
  • internal link : <a href="/index.html">Home Page</a>

More link attribute:

  • Download = lets users download files
  • Target = lets users specify a specific spot on the page that the link can open.
  • blank= opens in a new window/
  • top= opens at the top of the page
  • self= opens within the same iframe
  • parent= opens within the parent of the current frame
