Iframes

Iframes allow one HTML document to display the contents of one or more other HTML documents. We use the <iframe> element to accomplish this.

Iframes Syntax

<iframe src="url" title="description"></iframe>

The src attribute is used to determine the path to the html document or web page that will provide the contents of the iframe. Iframes are inline tags; if the width of an iframe is less than the width of the page, other content may appear to the right or left of it. The title attribute is used to provide screen readers (and in case the source page is inaccessible) a description of the page that should be displayed in the iframe.

You can use the height or width attribute to change the size of the window to your likeing.
