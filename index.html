What is HTML

HTML stands for hyper text markup launguage. This is the standard of devloping webpages. You can use html codes to develope the stucture of your webpage which consists of several elements. HTML elements instruct the browser on show to showcase the content. HTML is used to markup text instead of being a programming launguage such as JavaScript.

What is a element in HTML?

A HTML element is everything within a start tag and a end tag. A tag in HTML are keywords on a webpage that define how your web browser must formate and display your webpage. Since HTML Creates a format for text to be able to be displayed in a website and the markup is called tags which are showing in < and > symbols which we call angle brackets.

HTML codes usually look like this:

<start tag > html element < end tag >

There are 2 types of elements:

  1. Block Level elements
  2. Inline elements

Block level elements:

A block level element will automatically appear in a different line on the webpage.

<p> is a example of a block level element.

Inline elements:

Inline elements can appear next to one another horizontally so that text with an inline element in it will not starts a new line.

<span> is an example of a inline element.
