Simple HTML document

A example of a properly written wabepage:

  1. <!DOCTYPE html>
  2. <html>
  3. <head>
  4. <title> Page Title </title>
  5. </head>
  6. <body>
  7. <h1> My First Heading </h1>
  8. <p> My first paragraph </p>
  9. < /body >
  10. < /html >

Essential html elements every webpage needs:

In order to start a HTML code you need to start with the tag <!DOCTYPE html>. This shows that the document you are working on is a html5 document

The <html> element is the root element of an HTML page

The <head> element contains meta information about the HTML page

The <title> element specifies a title for the HTML page which is shown in the broswer's title

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

The <h1> element defines a large heading.There are 6 levels of headings. These headings define the different sizes of the headings for your webpage. The higher the number for the heading the smaller the heading will be. For example the size of the <h2> will be much arger then the heading size of <h5>.

The <p> element defines a paragraph

Formatting tags

  • <b>This will appear bold</b>
  • <strong>This will appear bold</strong>
  • <i>This will appear in italics<i>
  • <em>This will appear in italics</em>
  • <mark>This will appear highlighted</mark>
  • <small>This text will be smaller</small>
  • <del>This will appear crossed out</del>
  • <ins>This will appear underlined</ins>
  • <sub>This will appear as subscript</sub>
  • <sup>This will appear as superscript</sup>
