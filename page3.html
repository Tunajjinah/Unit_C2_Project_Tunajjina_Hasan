Tables and Lists in HTML

Lists elements : There are 2 types of lists

  1. Ordered list <ol> which is displayed in numbers.
  2. Unordered lists <ul> which is displayed as bullet points.

Thought they are 2 differengt lists they both use <li> to list items for whatever is within the list.

Structure of a basic ordered list :

<ol>

<li>list</li>

<li>is</li>

<li>numbered</li>

</ol>

Example of basic html list:

  1. Algebra 2
  2. Spanish 2
  3. 10th grade computer science
  4. 10th grade computer science

Structure of an unordered list :

<ul>

<li>list>/li>

<i>has</li>

<li>bullets</li>

</ul>

Example of a unordered list in html:

  • Alebra 2
  • Spanish 2
  • 10th grade computer science
  • 10th grade computer science

Borders for tables:

In order to create borders you need to use CSS.

We use the code "table, th, td { border:1px solid black; }"

By default the website will formate the table to fit the exact width of the webpage itself. However if you want to customize the website to your own liking then you can use "table { "width:100%; height:500px; } th,td { width:100px; height:50px;" }
